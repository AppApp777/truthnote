# TruthNote · 真笔记

**选中文字 → 右键 → 30 秒告诉你哪里有问题。**

一个浏览器插件，背后是一套**多 Agent 事实核查引擎**——不问大模型一句真假，而是真的联网到权威源取证、按规则给出**可追溯**的判断。

## 这是什么

朋友圈那条"存款超 5 万要交税"、短视频里那句"三天降血糖"、旧图配新文的"洪崖洞被淹"——**谣言换个说法就能反复翻炒，而普通人没有时间逐条去查。**

TruthNote 把"查证"这件事压进一次右键：在任意网页选中一句话，小 T（产品里的像素助手）会**真的上网**去找权威原文，几十秒后在原文上**标注**出每条声明是「查到了 / 不对 / 小 T 没查到」，每个判断都挂着可点开的出处。

它不替你删假信息，也不替你下结论——它**标出来，让你自己判断**。

## 核心亮点

- **真联网取证，不靠模型记忆。** 每条判断都来自联网调取的权威原文，而不是"大模型觉得"。证据链里每一条都能点开看出处。
- **自动路由到对口权威源。** 先分诊判断领域，再把声明送到最对口的地方核对：健康 → 卫健委 / WHO，政策 → 政府网 / 新华社，诈骗 → 反诈中心，并撞官方辟谣库。
- **一队 12 个分工 Agent + 零-LLM 规则裁决。** 分诊 / 拆解 / 筛选 / 常识 / 提问 / 取证 / 核对 / 评级 / 交叉核查 / 质疑 / 成卡 / 保健打假——分工协作，关键判定由可复现的规则引擎兜底，**可复现、可追责**。
- **可解释输出，不是黑箱单标签。** 二元判定（真 / 假）大字前置 + 证据链（每条挂出处）+ 6 维度信号分布，让你看见"为什么"。
- **内置官方辟谣库（万级条目）。** 命中只当作一条证据、判定仍重新核算，**绝不短路**——搜不到 ≠ 假。
- **社会闭环 · 公示墙。** 对还没定论的信息公开标注「待核实」，掐断"三人成虎"。
- **国产分层模型。** 底层 `deepseek-v4-flash` 跑海量初筛，顶层 `qwen-max` 做关键判定——是这套**分工架构**让便宜的国产模型也产出可靠结果。
- **360 生态联动（设想级可实现方案）。** 在 360 点睛广告投放前做虚假宣传预审，把"三天降血糖"这类功效夸大广告在投放前拦下——补一维核查，不替代平台审核。
- **多模态溯源。** 旧图新用反查更早出处，识别"移花接木"的配图谣言。

## 它是怎么工作的

```
                       ┌──────────────────────────────────────────┐
   选中一句话  ───────▶│              Orchestrator 编排器           │
                       └──────────────────────────────────────────┘
                                          │
   ① 声明提取 ─▶ ② 分诊路由 ─▶ ③ 证据搜索（联网/权威源/辟谣库）
        │                                 │
        ▼                                 ▼
   ④ 6 维度推理矩阵  ◀──  交叉核对  ──▶  ⑤ 反向质疑（时序识别 / 反诈）
        │                                 │
        └──────────────┬──────────────────┘
                        ▼
            ⑥ 零-LLM 规则引擎兜底（保守门 · 可复现）
                        ▼
            推理卡片：二元判定 + 证据链 + 6 维信号 → 在原文上标注
```

| 维度 | 信号含义 |
|---|---|
| **证实** | 是否有权威源正面支持 |
| **证伪** | 是否有权威源直接反驳 |
| **来源权威性** | 证据来自什么层级的源 |
| **时序一致性** | 旧闻翻炒 / 时间错位识别 |
| **内在逻辑** | 声明本身是否自洽 |
| **推销/保健话术** | FTC 式举证负担、功效夸大识别 |

> 判定体系对外呈现为**二元（真 / 谣言）**，仅是展示层投影；内部保留六值，`UNVERIFIED`（搜不到）走独立的"非裁定"通道，**绝不折叠进"谣言"**。

## 设计语言

TruthNote 的视觉基因：**纯矩形、直角、黑白主色，唯一的圆形是字母 o。** 助手"小 T"像 R2-D2——聪明、温暖、可靠，但明确是机器：它只有两只像素方块眼睛 `■ ■`，没有嘴，说话从不超过五个字（"查。" / "都对。" / "小 T 没查到。"）。

**色彩只有黑白 + 三判定色**，是全产品唯一的彩色：

| 判定 | 色值 | 含义 |
|---|---|---|
| 🟢 查到了 | `#3ECF72` | 有权威依据 |
| 🔴 不对 | `#F43939` | 被权威源反驳 |
| 🟡 小 T 没查到 | `#FFB020` | 没找到可靠来源（≠ 假） |

完整品牌规范见 [`前端/TruthNote-Design-Spec-v1.0.md`](前端/TruthNote-Design-Spec-v1.0.md)：Logo / 字标 / 小 T 表情系统 / 组件 / 动画 / CSS 变量一应俱全。

## 技术栈

| 层 | 技术 |
|---|---|
| 浏览器插件 | Manifest V3，原生 JS，覆盖微博 / 知乎 / 微信 / 小红书 / B 站 / X / 豆瓣 / 头条 / 贴吧十大平台适配器 |
| 后端 | Python · FastAPI（含流式取证 `/api/verify_stream`） |
| 引擎 | Orchestrator 编排 + 12 个 LLM Agent + 零-LLM 规则检测器 + 记忆层（官方辟谣库 `debunk_index`）+ 证据搜索 |

| 多模态 | 通义千问 VL 图片溯源 + 反向搜图 |

## 快速开始

```bash
# 1) 安装依赖
pip install -r requirements.txt

# 2) 配置模型 key：复制 .env.example 为 .env 并填入
cp .env.example .env

# 3) 启动后端（默认端口 8000）
python run.py

# 4) 加载插件：Chrome → 扩展程序 → 开启「开发者模式」
#    →「加载已解压的扩展程序」→ 选 前端/extension 目录
#    （或从 Releases 下载打包好的 zip 解压加载）

# 命令行调试单条
python cli.py "紧急通知！存款超5万要交税！"

# 跑测试集
python cli.py --file scenarios/rumor_testset.json --index 0
```

> 📦 预打包的插件见本仓 [**Releases**](../../releases) 页面，下载 zip 解压后即可"加载已解压的扩展程序"。
